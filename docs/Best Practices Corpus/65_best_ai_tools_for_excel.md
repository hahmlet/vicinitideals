# Best AI Tools for Excel in 2025
Source: https://www.adventuresincre.com/best-ai-tools-for-excel/
Reading Time: 9 min

AI tools for Excel are still very early. Most are far from making a meaningful impact on how analysts actually work. But the potential is real and will ultimately transform how we all work.

## Excel 4 CRE Add-in
Free, open-source VBA add-in built by A.CRE for CRE modeling. Adds real estate specific functions, finance-centric shortcuts, auto date headers, one-click DCF and waterfall starters, geocoding, residual land value helpers, auditing tools, and an AI Toolkit that uses Claude to explain or write formulas from plain English.

Fits analysts in acquisitions, development, asset management who want Excel-native, CRE-convention tools at zero license cost.

## FormulaBot
Spreadsheet-first AI environment with Excel and Google Sheets add-ins plus a Data Analyzer chat for multi-source analysis. Natural language to formulas, SQL, VBA, regex. Explains existing formulas, sentiment tagging, text extraction, PDF-to-table conversion.

## SheetGod
Plain English to Excel formulas, VBA macros, Google Apps Script, and regex. Task tutorials and automation helpers by BoloForms.

## Microsoft Copilot in Excel
Built into Microsoft 365. Analyzes tables, suggests formulas, generates charts, summarizes and models scenarios. Includes COPILOT function for cell-level AI. Best for organizations already on Microsoft 365.

## Formula Dog
Plain English to formulas, VBA, SQL, regex for Excel and Google Sheets. Reverse explains formulas into simple English.

## GPTExcel
Generates and explains formulas for Excel, Sheets, Airtable. Produces VBA and Apps Script, creates SQL and regex. Can chat over files, convert table images to spreadsheets, draft charts.

## Endex
Excel-native AI analyst backed by OpenAI's Startup Fund. Automates finance workflows like DCFs and three-statement models. Ingests SEC filings and internal docs with citations. Traces cell logic, builds reusable templates. Ties into Capital IQ or FactSet. Enterprise-focused with citation trails for auditability. Best for investment banking, PE, hedge funds.

## Shortcut
Autonomous Excel agent that builds models like DCF and LBO from prompts. Cleans data, analyzes PDFs, produces dashboards. Multi-agent plan-execute-verify loop for accuracy. Available as web app and Excel plugin.

## ChatGPT with Code Interpreter
Runs Python in a secure sandbox to read, clean, analyze, and write Excel or CSV. Creates charts, automates multi-step workflows. Mature Python tooling, simple upload-iterate UX.

## Claude Code with Sonnet 4.5
Embeds Claude in desktop or terminal workflow with secure Python sandbox and file access. Sonnet 4.5 adds high reasoning and long context for planning, code, and multi-file tasks, including creating and editing multi-sheet Excel workbooks from specifications.

## Relevance to Viciniti

Confirms the thesis: Excel is still dominant in CRE and AI is creeping in via copilots and code interpreters, not by replacing spreadsheets entirely. Viciniti's position as a non-Excel web app + Excel export is aligned with industry direction but not yet mainstream workflow.
