## Deviation notes and explanations

Observed drift on this baseline run is **zero**, so there are no unexplained discrepancies to triage today.

The retained non-zero tolerances are intentional and should remain in place for future runs because they absorb benign differences from:

1. **Currency rounding policy** — Excel display rounding vs Python `Decimal` quantization at persistence/export time.
2. **IRR / ratio math implementation details** — `pyxirr` solver behavior and DSCR annualization can differ from spreadsheet cell-by-cell formatting at very small decimal scales.
3. **Array-formula / period-boundary approximations** — workbook-style range formulas and month-boundary handling can introduce immaterial sub-cent or sub-basis-point noise in future fixture revisions.

Any future run that exceeds these thresholds should be treated as real model drift and investigated before promotion.
